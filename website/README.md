# DevelopersHub Corporation Website
A modern, fully responsive website build for the DevelopersHub Corporation internship task.

## Live Demo
[click here to view the website](#)

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts

## Pages
- Home
- About Us
- Services
- Portfolio
- Contact(with from validation)
- Booking From(3 step)

## Features
- Fully responsive design(mobile, tablet, desktop)
- Smooth hover animations
- Portfolio filter by category
- Contact form with validation
- 3-step meeting booking form
- Auto active navbar
- Mobile hamburger menu
